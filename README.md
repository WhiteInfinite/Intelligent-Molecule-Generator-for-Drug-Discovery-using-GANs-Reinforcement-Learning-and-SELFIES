# Intelligent-Molecule-Generator-for-Drug-Discovery-using-GANs-Reinforcement-Learning-and-SELFIES
1. Convert molecular structures into SELFIES representation.
2. Train a WGAN-GP model to learn molecular distributions.
3. Generate candidate molecules using the trained generator.
4. Apply RL to optimize molecular properties.
5. Evaluate molecules using QED, LogP and Synthesizability metrics.
6. Visualize valid molecules using RDKit.
